# LOOM 2.2.0 更新说明

## 总览体验

- 首次识别智能体安装状态后持久保存结果，切换页面和重启 LOOM 不再重复刷新。
- 安装、卸载、启动、重启和重新检测智能体后自动同步最新状态。
- 缓存数据损坏或格式升级时自动回退到一次安全检测。

## 原生智能体

- 强化手机目标范围约束，单台设备、设备组和全部在线设备不再互相串用。
- 同一会话已有可用图片或视频时优先复用，减少重复生成与重复计费风险。
- 修复失败任务中的重复工具卡片、残留准备回复和笼统错误提示。
- 手机、矩阵、媒体和组件任务日志增加可区分的中文诊断信息。

## 手机与矩阵

- 改进手机状态、截图和任务执行超时的错误归因，避免手机仍在执行却提前显示离线。
- 设备任务失败保留后端返回原因，便于直接判断连接、锁屏、无障碍或参数问题。
- 媒体传输遵循用户选择的设备范围，不再误传到会话范围之外的手机。

## 媒体生成

- 图片网关超时返回明确的可重试错误，避免拆分请求造成重复生成。
- 支持复用会话内已有生成结果继续执行传输和发布流程。
- 发布任务缺少标题时可从用户输入中安全补全必要字段。

## 安装与运行

- 延续受保护运行时、中文交互和标准 NSIS 安装体验。
- 保留既有用户配置与数据，支持从 2.1.x 直接升级。
