# LOOM 2.1.108 更新说明

## Agent 接入与 Skill 分发

- Agent 接入改用 `loom.heang.top` 国内主源下载统一 Skill，GitHub 固定版本地址降级为备用源。
- 保留安装包内置 Skill 作为离线兜底，形成国内主源、GitHub 备用、安装包内置三层下载策略。
- 国内主源使用带哈希的不可变文件名，避免 CDN 旧缓存返回过期 Skill。
- 三个来源统一使用同一 SHA256 校验，文件不一致时立即停止安装。
- Agent 接入高级配置同时展示主源、备用源和校验值，便于诊断下载问题。

## 云端发布

- 已将 `2026.07.20` 统一 Skill Library 原子发布到麓鸣下载站。
- 云端 ZIP 与安装包内置 ZIP 的大小及 SHA256 完全一致。
