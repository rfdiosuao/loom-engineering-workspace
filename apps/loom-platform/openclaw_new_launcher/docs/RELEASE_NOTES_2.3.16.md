# LOOM 2.3.16 更新说明

## 智能体品牌资产

- 修复中枢智能体页顶部 Logo 和新对话空状态 Logo 显示为破损图片的问题。
- 智能体、思考状态和空状态统一复用已纳入安装包的麓鸣 SVG 品牌标识。
- 移除 4 个未随应用发布的历史 PNG 资源引用，避免在桌面安装包中出现失效路径。
- 品牌图片加载异常时自动使用内置矢量标识回退，不再向用户显示浏览器破图图标。

## 发布验证

- 新增品牌运行时资源完整性测试：`/loom-motion/*` 静态引用必须对应实际随包资源。
- 前端生产构建会生成带指纹的 `luming-logo` 资源，确保桌面包使用确定的本地文件。
