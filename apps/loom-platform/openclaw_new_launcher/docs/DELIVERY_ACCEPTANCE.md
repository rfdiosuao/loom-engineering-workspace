# OpenClaw 交付验收清单

> 适用版本：OpenClaw Portable / 桌面启动器当前交付版

## 1. 交付范围

- 启动器可正常启动
- 授权流程可正常完成
- 微信插件可安装、可扫码绑定
- 飞书插件可安装、可填写 App ID / App Secret 并绑定
- 服务日志可查看、可滚动、可导出
- AI 生图可用
- AI 视频可用并可预览/下载
- 浅色 / 深色主题切换正常
- 环境诊断可一键修复常见启动问题

## 2. 验收前检查

- [ ] 启动器未被其他实例占用
- [ ] 18790 / 18791 等本地端口未冲突
- [ ] 本机网络可访问 OpenClaw 相关服务
- [ ] Node.js / 依赖目录完整
- [ ] 包体内不包含客户不该看到的源码、临时文件、测试残留

## 3. 必测项

### 3.1 启动与授权

- [ ] 启动器打开后不白屏、不报错
- [ ] 授权码输入后可激活
- [ ] 激活状态刷新后保持正确
- [ ] 未授权状态下仅限制应限制的功能

### 3.2 微信插件

- [ ] 点击安装后不死锁
- [ ] 日志窗口能看到安装命令和扫码提示
- [ ] 弹出二维码完整可见
- [ ] 扫码后能完成绑定
- [ ] 不会再重复弹出新启动器

### 3.3 飞书插件

- [ ] 点击安装后有命令行输出
- [ ] 二维码 / 登录链接完整显示
- [ ] App ID / App Secret 可手动填写
- [ ] 保存后配置生效

### 3.4 AI 能力

- [ ] AI 生图能提交任务
- [ ] AI 生图返回结果可预览
- [ ] AI 视频能提交任务
- [ ] AI 视频生成结果可在页面展示并下载
- [ ] 下载后有路径提示

### 3.5 日志与主题

- [ ] 日志可滚动查看历史内容
- [ ] 导出日志可生成文件
- [ ] 打开目录按钮可正常打开日志目录
- [ ] 深色 / 浅色切换按钮正常工作
- [ ] 启动时不会先闪现错误主题

## 4. 交付物检查

- [ ] 安装包 / 压缩包命名符合版本号
- [ ] README / 使用说明存在
- [ ] 售后排障说明存在
- [ ] Mac 迁移说明存在（如需要）
- [ ] Gitee 标签已打好

- 已知限制：
