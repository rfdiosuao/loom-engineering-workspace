# OpenClaw 客户使用说明

## 1. 产品简介

OpenClaw 是一个面向业务交付的桌面启动器，用来统一管理授权、插件绑定、AI 生图、AI 视频、日志查看与环境修复。

## 2. 首次使用

1. 打开启动器。
2. 输入授权码并完成激活。
3. 在“API 配置”里填入生图 / 视频需要的密钥。
4. 在“微信机器人”里点击安装并扫码绑定。
5. 在“飞书机器人”里点击安装并按提示填写应用信息。
6. 如果是新装的 Windows 10 / 11 且启动失败，先打开“环境诊断”，检查 WebView2、Python、Node、端口占用和残留进程。

## 3. 常用功能

### 3.1 服务日志

- 用来查看启动、安装、绑定、请求和错误信息。
- 可以上下滚动查看历史日志。
- 可以导出日志给售后排查。

### 3.2 AI 生图

- 填写提示词后提交任务。
- 等待生成完成后可预览结果。
- 如需替换模型或密钥，请进入 API 配置。

### 3.3 AI 视频

- 支持文生视频 / 图生视频。
- 提交后等待任务完成。
- 结果完成后可预览与下载。

### 3.4 微信机器人

- 微信机器人只支持扫码绑定，不支持手动填写 ID / Key。
- 点击安装后，右侧日志区会显示命令和二维码。
- 用微信扫码即可完成绑定。

### 3.5 飞书机器人

- 飞书机器人支持扫码或链接绑定，也支持手动填写 App ID / App Secret。
- 点击安装后，右侧日志区会显示安装命令和二维码 / 链接。
- 如二维码不完整，可复制链接到浏览器打开。

### 3.6 环境诊断

- 该功能用于检查常见环境问题。
- 可以直接查看 WebView2、Python、Node、端口占用、残留进程和关键配置是否正常。
- 如果服务启动失败，先看这里，再导出日志给售后。

## 4. 主题切换

- 支持浅色 / 深色两种风格。
- 可以通过右上角主题按钮切换。

## 5. 日常建议

- 不要同时启动多个同版本实例。
- 每次更新后先检查插件安装状态。
- 如果功能异常，优先导出日志给售后。
