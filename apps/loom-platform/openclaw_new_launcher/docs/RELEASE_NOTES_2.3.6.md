# LOOM 2.3.6 更新说明

## 智能体工具调用

- 将后台任务、系统状态、模型、账号、授权、诊断、定时任务和 Agent 管理能力改为按明确意图开放，减少无关工具调用和过程刷屏。
- 手机、矩阵、媒体、获客与飞书集成的动作进一步拆分，查询、修改、下发、取消、重试和发布不再互相误触发。
- 原生生图、生视频、素材传输和手机发布已纳入生产能力目录回归，确保连接态能力也经过路由验证。

## 自然语言理解

- 支持“抖音自动拓客”“招聘自动化”“找客户”等业务口语直接进入获客任务。
- 修复 BOSS 直聘简历筛选误带手机发布工具的问题；只有用户明确要求发布时才开放发布能力。
- “你好”“你是谁”“介绍一下你自己”等普通对话不再加载完整工具目录。

## 稳定性保护

- 生产能力目录逐项验证工具名称、目标权限和路由边界，并保留必要的状态预检组合。
- 产品版本统一升级到 `2.3.6`，安装包、前端、Rust 与发布元数据保持一致。
