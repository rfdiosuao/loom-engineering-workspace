# LOOM 2.3.18 更新说明

## Agent 会话保护

- 修复重新配置 Codex 或 Claude Code 后原有历史会话可能不可见的问题。
- 保留 Codex 默认 `%USERPROFILE%\.codex` 和用户已有的自定义 `CODEX_HOME`。
- 切换中转站、API 地址、模型或账号配置时，活动会话与归档会话继续保留。
- 配置前后自动核对会话数量、会话索引和状态数据库；发现异常时自动回滚配置。
- 安装器升级与启动器写入配置时不再强制把 Codex 切换到 LOOM 私有会话目录。

## 可见状态与兼容性

- Agent 安装页面显示原有会话保护状态、识别数量和最近校验结果。
- 兼容已经存在的 Codex 官方登录、MCP、插件和个人配置。
- 不迁移、不修改会话正文，也不会按中转站或模型筛选历史会话。
