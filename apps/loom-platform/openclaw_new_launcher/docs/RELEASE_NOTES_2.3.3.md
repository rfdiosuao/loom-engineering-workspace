# LOOM 2.3.3 更新说明

## 智能体工具调用

- 已有图片或视频传到手机相册时，智能体优先复用本地素材，不再误触发重复生成。
- 传到手机相册与发布到内容平台分离，普通素材传输不再误调用发布能力。
- 打开手机设置等设备操作不再混入 LOOM 主题、更新和系统配置能力。
- 招聘海报等创作场景不再误触发招聘获客流程；明确要求获客时仍可正常调用。

## 能力收敛

- 普通生图、生视频只开放执行能力，隐藏 API 配置和诊断测试工具。
- 普通手机控制隐藏 ADB 修复、事件同步和发布工具，只有明确诊断或同步请求才会启用。
- 显式能力提示和核心安全能力保持可用，历史调用契约无需迁移。

## 稳定性

- 新增自然语言意图隔离回归测试，覆盖素材复用、相册传输、手机设置、媒体配置和手机诊断。
- 产品版本统一升级到 `2.3.3`，安装包、前端与 Rust 元数据保持一致。
