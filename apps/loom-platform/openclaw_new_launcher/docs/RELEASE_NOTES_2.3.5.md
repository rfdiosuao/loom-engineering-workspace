# LOOM 2.3.5 更新说明

## 智能体意图理解

- 新增中文否定语义识别。用户明确说“不要”“无需”“禁止”或“请勿”时，被否定的工具不再进入本轮能力集合。
- 支持在同一句话中区分正向目标与排除项，例如“只生成视频，不要生成图片”和“取消矩阵任务，不要重试”。
- 修复招聘海报、获客任务等跨领域表达互相误触发的问题，减少无关工具调用和执行过程刷屏。

## 手机与矩阵路由

- 修复“修复 ADB 连接，但不要执行手机任务”被错误路由到全量能力目录的问题。
- ADB、无障碍和手机连接修复会保留手机领域能力，同时排除被明确否定的执行动作。
- 矩阵查看、取消、重试和下发动作继续按用户当次意图独立开放。

## 回归保护

- 增加媒体生成、素材传输、矩阵任务、系统更新、手机事件和媒体配置等否定意图回归测试。
- 产品版本统一升级到 `2.3.5`，安装包、前端、Rust 与发布元数据保持一致。
