# LOOM 2.1.91 更新说明

## 矩阵执行与截图

- 矩阵任务下发接入统一 Job 轮询，界面会持续显示等待、执行、完成或失败状态。
- 矩阵截图复用手机连接模块的截图解析器，兼容单行及格式化多行 JSON 返回。
- 矩阵默认采用安全执行模式，避免未确认情况下扩大任务权限。

## 智能体能力

- 按业务操作去重内部能力、CLI 和 MCP 能力，智能体只看到一个可执行入口。
- 保留旧能力 ID 兼容映射，历史会话和模型工具调用无需迁移。
- 补齐手机状态、手机事件、获客智能体、飞书集成和环境诊断等 MCP 能力。
- CLI 与 MCP 能力目录实现 79 项一一对应，并完成 79 项真实干跑验证。
- 修复诊断修复参数、矩阵目标约束、模板设备约束和定时任务干跑行为。

## 手机安全链路

- 删除渲染层可携带明文手机令牌的旧 Tauri 代理入口。
- 手机、矩阵和智能体继续统一经过 Python Bridge、受控子进程及 APKClaw 执行链。
- 删除重复的手机模型同步实现，保留包含设备下发的完整实现。
