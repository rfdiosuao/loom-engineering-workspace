# 更新日志 v2.0.6

日期：2026-05-23

## 本次改动
- 补齐 `npm run phone:agent` 和 `npm run phone:fleet`，OpenClaw 可以直接走 launcher 包装器调用 APKClaw。
- `openclaw-context` 补齐 `phoneFleetCli`、`fleetCli`，并把 `OPENCLAW_MEMORY_RECOVERY.md` 纳入启动上下文。
- 打包脚本同步便携包 `package.json` 的 `name/version/scripts`，避免包内脚本缺失。
- 修复便携包 JSON 写入 BOM 问题，避免新电脑首次解压后被严格解析误判为损坏。
- `openclaw.json` 和 `runtime-context.json` 读取兼容 UTF-8 BOM。
- 新增 `OPENCLAW_MEMORY_RECOVERY.md`，用于 OpenClaw 丢失上下文后的能力恢复。

## 新电脑验证
- 全新路径解压验证通过。
- `phone:agent` 可执行。
- `phone:fleet` 可执行。
- `openclaw:context --json` 可读到恢复文档、群控入口和 wrapper-only 约束。
- `runtime-context.json`、`openclaw.json` 解析正常。

## 以后规则
- 以后每次改动都新增一份同格式版本日志文件。
- 文件名统一使用 `YYYY-MM-DD-vX.Y.Z-更新日志.md`。
