# LOOM 2.3.2 更新说明

## 智能体素材复用

- 智能体现在可以查询本地素材库中的已有图片和视频。
- 用户提到“之前生成”或“已有素材”时，优先复用原素材，不再无故重复生成。
- 支持把指定素材传到明确选择的一台、多台、设备组或全部在线手机。
- 素材传输返回真实任务终态，失败时保留可诊断原因，不再显示假成功。

## 能力路由

- “全部设备”“所有手机”“每台手机”等表达会正确进入矩阵分发能力。
- 安装或检测 Codex、Claude Code、OpenClaw、OpenCode、Hermes 时会聚焦智能体安装能力。
- “明天九点”“今晚八点”“两小时后”等自然时间表达会正确进入定时任务能力。

## 稳定性

- 修正 Windows 环境下手机事件流测试的固定时延竞争，改为验证事件确实早于任务终态。
- 产品版本统一升级到 `2.3.2`，安装包、前端与 Rust 元数据保持一致。
