# LOOM / 麓鸣白标定价与销售材料规划

日期：2026-07-05

本文档用于沉淀 LOOM / 麓鸣白标与私有化交付的定价结构、套餐设计、持续收费方式和后续销售材料清单。它配合 `LOOM_WHITE_LABEL_BUSINESS_PLAN.md` 使用。

## 1. 定价原则

LOOM 白标方案不建议按“一个软件多少钱”来卖，而应按“交付 + 授权 + 运维 + 用量”四层收费。

原因：

1. 客户买到的不是单个启动器，而是一套可销售、可运营的 AI 自动化平台。
2. 白标交付包含中转站、模型配置、安装器、Agent、手机矩阵、发布链路和培训。
3. 后续会持续产生模型消耗、模板定制、版本更新和运维支持。
4. 如果只按一次性软件价出售，后期售后成本和版本维护会被低估。

推荐结构：

```text
首期交付费
  + 年度授权费 / 年维护费
  + 模型用量或充值分成
  + 模板和流程定制费
  + 运维与升级服务费
```

- 基础品牌色替换。

## 3. 持续收费设计

### 3.1 年维护费

建议：

- 按首年交付价的 `20% - 30%` 收取。
- 或按固定套餐收取。

示例：

| 首期交付价 | 年维护费建议 |
| --- | --- |
| ¥19,800 | ¥4,800 - ¥6,800 / 年 |
| ¥39,800 | ¥9,800 - ¥12,800 / 年 |
| ¥98,000 | ¥19,800 - ¥29,800 / 年 |
| ¥150,000+ | 单独报价 |

维护内容：

- 白标包兼容更新。
- 中转站基础问题排查。
- 模型同步适配。
- 安装器更新。
- 常见 Bug 修复。
- 发布清单维护。
- 基础远程支持。

不包含：

- 新模块开发。
- 深度行业模板。
- 客户服务器长期代运维。
- 大规模人工客服。

### 3.2 模型用量分成

可选方式：

1. 按充值金额分成。
2. 按模型消耗差价分成。
3. 按代理等级给批发价。
4. 按月结算。

建议口径：

> 模型成本随上游渠道变化，最终价格以实际模型通道、倍率和客户消耗为准。

不要承诺：

- 永久固定低价。
- 所有模型无限调用。
- 上游永不失效。

### 3.3 模板定制费

建议价格：

| 模板复杂度 | 价格区间 |
| --- | --- |
| 简单模板 | ¥3,000 - ¥8,000 / 个 |
| 中等模板 | ¥8,000 - ¥20,000 / 个 |
| 复杂行业模板 | ¥20,000 - ¥50,000+ / 个 |

示例模板：

- 小红书发布模板。
- 电商商品巡检模板。
- 本地生活素材发布模板。
- App 测试巡检模板。
- 客服辅助模板。
- 素材生成到手机发布模板。

### 3.4 远程部署支持费

建议价格：

- 单次远程排障：`¥500 - ¥2,000`
- 服务器部署支持：`¥2,000 - ¥8,000`
- 客户现场或深度交付：单独报价

### 3.5 版本升级服务费

建议价格：

- 小版本升级：维护期内包含，或 `¥1,000 - ¥3,000 / 次`
- 大版本升级：`¥5,000 - ¥20,000 / 次`
- 白标大改版：单独报价

### 3.6 私有化运维费

建议价格：

- 轻量运维：`¥3,000 - ¥5,000 / 月`
- 标准运维：`¥8,000 - ¥15,000 / 月`
- 专属运维：`¥20,000+ / 月`

## 4. 推荐首批成交方案

第一阶段不建议一开始就卖过高价格。推荐先用“白标专业版”打样成交。

推荐报价：

```text
白标专业版：¥39,800
首年基础维护：包含 3 个月
次年维护费：¥9,800 / 年
模型用量：另算
行业模板：按需报价
```

这个价格的好处：

1. 对有客户资源的小团队来说不算离谱。
2. 能覆盖换壳、部署、联调和培训成本。
3. 给后续续费和定制留空间。
4. 不会把产品贬低成一次性小工具。

## 5. 报价边界

报价时必须明确“包含”和“不包含”，避免客户无限加需求。

### 5.1 默认包含

- 一个品牌客户端。
- 一个中转站接入。
- 一个发布通道。
- 一个默认模型策略。
- 一套基础安装器。
- 基础手机控制。
- 基础创作入口。
- Agent 接入说明。
- 基础培训。
- 基础验收。

### 5.2 默认不包含

- 多品牌管理。
- 多代理后台。
- 支付系统开发。
- 企业权限体系。
- 行业深度流程。
- 大规模手机矩阵专项调试。
- 客户业务数据清洗。
- 长期人工代运营。
- 平台风控绕过。
- 验证码和支付自动化。

### 5.3 高风险承诺禁止

不要承诺：

- 保证涨粉。
- 保证成交。
- 保证账号不封。
- 保证绕过平台规则。
- 保证所有手机都稳定。
- 保证所有 Windows 环境无安全提示。
- 保证所有模型永远可用。

可以承诺：

- 提供稳定可回滚的白标客户端。
- 提供模型账号同步能力。
- 提供智能体安装和配置入口。
- 提供手机 Agent 自动化底座。
- 提供 CLI/MCP 接入能力。
- 提供日志、诊断和基础运维。

## 7. 销售材料制作优先级

建议按这个顺序做：

1. 一页纸报价单。
2. 白标资料收集表。
3. 交付验收清单。
4. 代理合作方案。
5. 客户演示脚本。
6. 销售 PPT。
7. 官网 / 落地页文案。
8. 合同条款草案。

原因：

- 报价单能立刻用于沟通。
- 资料收集表能立刻用于成交后推进。
- 验收清单能保护交付边界。
- PPT 和官网适合稍后包装。

## 8. 推荐销售口径

### 8.1 简短版

> 我们不是只帮你搭一个中转站，而是帮你交付一套自己的 AI 自动化产品：有你的品牌、有你的安装器、有你的模型账号、有手机矩阵和 Agent 接入能力。你可以直接拿去给客户卖。

### 8.2 代理版

> 你负责客户和渠道，我们负责中转站、白标客户端、模型接入、安装器、Agent 能力和更新维护。你可以用自己的品牌销售 AI 自动化产品，而不是只转卖 API。

### 8.3 企业版

> 这是一套企业 AI 自动化工作台。员工登录账号后，可以安装 AI 工具、连接手机、生成素材、执行任务模板和查看日志，把重复工作逐步交给 AI。

### 8.4 投资人 / 合作方版

> LOOM 切的是 AI Agent 落地最后一公里。模型能力已经成熟，但企业缺少安装、配置、调度、监控和白标交付入口。我们把中转站、桌面客户端、手机自动化、Agent 接入和任务模板打包成一套可复制交付的商业系统。
