## 发现的问题

1. Agent 对闲鱼 WebView 复合页面识别不稳定，会把 `我发布的` 下方推荐转卖列表误判为发布管理列表。
2. 长 prompt 容易误触 `发闲置` 编辑页；闲鱼自动化应改为短步骤状态机和截图坐标校验。
3. 一键擦亮没有可验证成功态，启动器不能直接报“成功”，应显示“已点击，等待平台反馈”。
4. 当前服务进入超级曝光页后显示无可加曝光宝贝，可能是服务类不满足平台曝光条件，需用普通商品复测。
5. `赚曝光` 面板中的 `去发布商品` 和 `闲鱼币兑换曝光` 都会改变真实状态或资源，必须作为高风险动作停在确认前。
6. Agent 退出发布表单后文字总结把页面说成首页，但节点树和截图显示仍在 `赚曝光` 面板；报告以截图/节点树为准。

目前“手机控制桥 + 闲鱼入口执行 + dry-run 安全边界”已经跑通；但还不能算完整交付。
