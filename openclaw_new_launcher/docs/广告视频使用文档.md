# AI 广告工作台使用文档

> 适用功能：启动器左侧 `广告视频` 模块  
> 当前定位：分镜、三视图、九宫格、首尾帧和图生视频工作台  
> 适用版本：Lumi / OpenClaw Launcher v2.0.1+

## 1. 功能定位

`广告视频` 不是“一键自动剪辑成片”工具，而是面向短视频广告创作者的镜头生产工作台。它帮助用户把一个产品广告拆成多个镜头，每个镜头先生成九宫格候选图，再挑选首帧和尾帧，最后生成单个镜头视频。

推荐使用方式：

```text
上传产品图 -> 写镜头卖点 -> 生成九宫格 -> 选首帧/尾帧 -> 生成镜头视频 -> 质检 -> 保存项目
```

如果一条广告需要 8 个镜头，就创建 8 个分镜；每个分镜都可以独立生成九宫格和视频。

## 2. 使用前准备

### 2.1 授权

广告视频属于授权功能。用户需要先在 `授权码` 页面完成在线激活。

### 2.2 AI 生图配置

九宫格候选图依赖 `AI 生图` 配置。生成九宫格前，请先确认：

- `AI 生图` 页面已经填写 `Url链接`
- API Key 可用
- 生图接口支持图片生成或图片编辑

如果没有配置，点击 `生成九宫格` 会提示先配置生图服务。

### 2.3 视频服务配置

每次生成视频时，需要在右侧 `视频服务商` 区域填写视频服务密钥。

当前支持：

| 服务商 | 用途 | 说明 |
| --- | --- | --- |
| 阿里云 DashScope / 快乐马 | 默认视频服务 | 适合快速接入和稳定交付 |
| 火山引擎 Seedance | 可选视频服务 | 适合扩展更多视频模型 |
| 自定义兼容服务 | 私有网关或兼容服务 | 需要填写 API Base URL 和模型 ID |

注意：广告工作台里的视频 API Key 仅本次使用，页面不会保存。客户每次重新打开后需要重新填写。

## 3. 页面区域说明

### 3.1 顶部栏

顶部显示：

- 页面标题：`AI 广告工作台`
- 当前质检进度：例如 `质检 3/5`
- `保存项目` 按钮

保存项目会保存分镜、提示词、九宫格候选图、首尾帧和已生成视频记录。

### 3.2 左侧 Scenes 镜头列表

左侧用于管理分镜：

| 操作 | 说明 |
| --- | --- |
| 新增 | 新建一个空镜头 |
| 复制 | 复制当前镜头，用于快速复用卖点、比例、运镜等设置 |
| 删除 | 删除当前镜头；至少会保留一个镜头 |
| 点击镜头 | 切换到对应镜头进行编辑 |

建议命名方式：

```text
01 开场吸引
02 产品外观
03 核心卖点
04 使用场景
05 细节特写
06 效果对比
07 信任背书
08 结尾行动
```

### 3.3 产品素材区

产品素材区包含两类参考图。

| 区域 | 作用 |
| --- | --- |
| 上传产品图 | 当前镜头专用参考图，优先级最高 |
| 三视图备用参考 | 全项目通用参考图，包含正面、侧面、背面 |

推荐规则：

- 如果当前镜头需要严格保持某一张产品图，使用 `上传产品图`
- 如果多个镜头都要保持同一个产品外观，先上传正面、侧面、背面三视图
- 产品图要清晰、主体完整、背景干净，避免裁切掉包装、Logo 或重要结构

### 3.4 首帧 / 尾帧区

| 名称 | 作用 |
| --- | --- |
| 首帧 | 视频开始画面，生成视频前必须设置 |
| 尾帧 | 结尾画面控制，用于让提示词约束结尾构图和情绪 |

当前版本的图生视频接口主要使用首帧作为真实输入。尾帧会进入视频提示词，用于辅助控制结尾方向，但不同视频服务商对尾帧的执行效果可能不同。

设置首尾帧的方式：

- 直接上传图片
- 在九宫格里选中候选图，再点击 `设为首帧`
- 在九宫格里选中候选图，再点击 `设为尾帧`

### 3.5 九宫格创作区

九宫格用于一次生成 9 张候选关键帧。工作流是：

1. 填写 `九宫格提示词`
2. 点击 `生成九宫格`
3. 从 9 张候选图中选择最适合的一张
4. 设置为首帧、尾帧，或作为下一轮参考图

按钮说明：

| 按钮 | 作用 |
| --- | --- |
| 生成九宫格 | 根据当前产品参考图和九宫格提示词生成 9 张候选图 |
| 清空候选 | 清空当前镜头的候选图 |
| 设为首帧 | 把选中的候选图设为当前镜头首帧 |
| 设为尾帧 | 把选中的候选图设为当前镜头尾帧 |
| 作为下一轮参考图 | 用选中候选图继续二次生成，适合逐步修正画面 |

## 4. 镜头设置说明

右侧 `Scene Settings` 控制当前镜头。

| 字段 | 建议写法 |
| --- | --- |
| 镜头标题 | 简短描述镜头目的，例如 `产品开场`、`使用场景` |
| 核心卖点 | 一句话说明这个镜头要让观众记住什么 |
| 时长 | 当前支持 5 秒或 10 秒 |
| 比例 | 常用短视频建议 `9:16`，横版素材可选 `16:9` |
| 运镜 | 选择缓慢推进、横向平移、环绕展示、静物特写或拉远收束 |
| 视频提示词 | 描述镜头运动、产品状态、场景和情绪 |
| 负面词 | 写清楚要避免的画面问题 |

推荐负面词示例：

```text
低清晰度，变形，杂乱背景，错误文字，遮挡产品，品牌错误，产品结构变化，Logo 模糊。
```

## 5. 标准创作流程

### 5.1 新建项目

首次进入 `广告视频` 页面后，系统会默认创建一个镜头。用户可以直接编辑，也可以点击 `新增` 创建更多镜头。

### 5.2 上传产品参考

建议先上传三视图：

1. 点击 `正面`
2. 上传产品正面图
3. 点击 `侧面`
4. 上传产品侧面图
5. 点击 `背面`
6. 上传产品背面图

如果某个镜头需要特殊参考，再点击 `上传产品图` 上传本镜头专用图片。

### 5.3 写九宫格提示词

九宫格提示词建议包含：

- 产品主体
- 背景环境
- 光线风格
- 卖点表达
- 构图要求
- 投放平台风格

示例：

```text
一只便携 U 盘启动器放在干净桌面中央，产品主体完整清晰，金属质感明显，背景简洁明亮，画面突出“插上即可启动 AI 服务”的卖点，适合抖音信息流广告开场关键帧。
```

### 5.4 生成九宫格并选择关键帧

生成后优先选择满足以下条件的图：

- 产品外观没有变形
- Logo 或包装文字没有明显错误
- 主体完整，没有被裁切
- 背景不抢戏
- 一眼能看懂卖点
- 符合当前视频比例

选中候选图后，按需要点击：

- `设为首帧`
- `设为尾帧`
- `作为下一轮参考图`

### 5.5 生成视频

生成视频前必须确认：

- 当前镜头已经设置首帧
- 视频服务商已选择
- API Key 已填写
- 模型已选择或手动输入
- 视频提示词、负面词已检查

点击 `生成视频` 后等待任务完成。生成成功后页面会显示播放器，并在状态文字中显示视频大小和保存路径。

生成的视频默认保存到：

```text
OpenClawFiles/data/videos/
```

开发环境下路径以当前项目运行目录为准。

### 5.6 质检并保存

每个镜头生成后，使用右侧质量检查：

| 检查项 | 通过标准 |
| --- | --- |
| 产品不变形 | 产品外观、比例、结构稳定 |
| Logo / 包装清晰 | Logo、包装文字、品牌元素没有明显错误 |
| 卖点一眼可懂 | 观众 1 到 3 秒内能理解镜头意图 |
| 首尾帧连贯 | 起始画面和结尾画面运动自然 |
| 适合投放裁切 | 在目标比例下主体不被裁掉 |

全部检查后点击 `保存项目`。

## 6. 怎样判断视频顺不顺

从创作者视角看，顺不顺主要看 5 件事：

1. 镜头目的是否单一  
   一个镜头只讲一个卖点，不要同时讲外观、价格、功能和活动。

2. 主体是否稳定  
   产品不要忽大忽小、结构变化、Logo 漂移或包装文字乱跳。

3. 运镜是否服务卖点  
   外观展示适合缓慢推进或环绕展示；细节卖点适合静物特写；结尾 CTA 适合拉远收束。

4. 首尾是否能接上  
   如果上一个镜头结尾是产品特写，下一个镜头开头不要突然变成完全无关的场景。

5. 投放比例是否安全  
   竖版广告优先使用 `9:16`，确认产品主体、字幕区域和 Logo 没有贴边。

## 8. 常见问题

### 8.1 为什么不能生成九宫格？

通常是 `AI 生图` 页面没有配置 `Url链接`，或 API Key 不可用。先到 `AI 生图` 页面配置并测试。

### 8.2 为什么不能生成视频？

常见原因：

- 当前镜头没有设置首帧
- 视频 API Key 为空
- 自定义服务没有填写 API Base URL
- 模型 ID 为空或服务商不支持该模型
- 视频服务商任务超时或返回的视频地址不可下载

### 8.3 为什么尾帧效果不稳定？

当前版本主要通过提示词约束尾帧，不同视频模型对尾帧控制能力不同。若尾帧不稳定，建议把尾帧画面描述写进 `视频提示词`，并在下一轮九宫格中继续修正。

### 8.4 为什么产品外观变了？

可以按顺序处理：

1. 上传更清晰的产品图
2. 补充正面、侧面、背面三视图
3. 在负面词中加入 `产品结构变化，Logo 错误，包装文字错误`
4. 使用 `作为下一轮参考图` 逐轮修正
5. 换用更适合图生视频的模型

### 8.5 九宫格生成后是否自动变成 8 个镜头？

不会。九宫格是候选关键帧，不是自动分镜。用户需要手动创建镜头，并为每个镜头选择合适的首帧、尾帧和视频提示词。
