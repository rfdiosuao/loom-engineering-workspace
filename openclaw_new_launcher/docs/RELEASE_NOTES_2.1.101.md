# LOOM 2.1.101 更新说明

## 多手机管理

- 修复修改一台手机名称后，矩阵工作台所有手机名称一起变化的问题。
- 手机配置成为设备名称、分组和身份信息的唯一来源，运行时状态不再覆盖设备名称。

## 素材传输

- 本地素材库新增“传到手机”，可以选择一台或多台目标手机。
- 新生成的图片和视频会并行传输到全部已配置手机。
- 每台手机保留独立传输结果，部分设备失败不会掩盖其他设备的成功结果。

## 矩阵任务

- 智能体和矩阵工作台会显示失败设备、错误码及具体失败原因。
- 任务下发提交后立即恢复界面响应，执行进度在后台持续更新。
- 急停改为立即发送取消信号并返回，不再等待后台线程收尾。
- 修复旧任务结果覆盖新任务提示的问题。

## 验证

- 后端矩阵、手机、素材和任务测试通过。
- 前端平台与 Node 契约测试通过。
- 960x640、1200x800、1440x900 三档桌面视口 E2E 通过。
- 前端生产构建通过。
