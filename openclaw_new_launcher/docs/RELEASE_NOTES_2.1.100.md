# LOOM 2.1.100 更新说明

## 环境诊断

- 移除已经不参与当前启动链的 OpenClaw 启动入口、本体和基础配置检查。
- 环境诊断不再因为旧组件缺失显示无效阻塞项。

## 系统日志

- 原“LOOM 运行时”页面升级为真实系统日志页，默认聚合错误与警告。
- 支持切换查看全部 Bridge 运行日志，并保留导出、定位和清空功能。
- 日志采集不再依赖 OpenClaw 服务状态，服务停止时仍能查看真实报错。

## 图片生成

- 图片生成默认模型统一为 `gpt-image-2`。
- 前端初始值、配置快照、保存配置和实际生成链使用同一默认模型。

## 智能体会话

- 修复归档会话仍残留在左侧列表，必须切换页面后才消失的问题。
- 归档成功后立即更新当前会话和搜索结果。

## 验证

- 58 项相关 Python 测试通过。
- 119 项前端平台合同测试通过。
- 系统日志桌面 E2E 与前端生产构建通过。
