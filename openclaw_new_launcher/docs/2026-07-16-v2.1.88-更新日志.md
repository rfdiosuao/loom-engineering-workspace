# 麓鸣 2.1.88 更新日志

发布日期：2026-07-16

## 升级修复

- 修复从旧版本覆盖升级后，残留 Python 源码遮蔽受保护模块，导致中枢智能体和矩阵手机画面接口返回“接口不存在”的问题。
- 安装前仅清理 LOOM 管理的 Python 与脚本目录，继续保留 `data` 内的账号、配置和业务数据。
- 新增旧版残留覆盖升级烟测，验证智能体、矩阵画面路由来自当前受保护模块，同时验证用户数据不丢失。

## 错误诊断

- 中枢智能体页面现在显示 Bridge 返回的真实结构化错误，不再统一替换成“运行时信息读取失败”或“会话列表读取失败”。
- 未授权、后端未启动或接口异常时，可以直接根据错误内容定位问题。
